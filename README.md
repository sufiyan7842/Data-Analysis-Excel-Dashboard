# Monthly Salary Dashboard - Excel Analytics Project

## Project Overview
This project is an interactive **Excel Dashboard** created to analyze monthly business salary and sales performance using raw transactional data. The dashboard converts unstructured data into meaningful insights through KPI cards, charts, summary tables, and performance tracking.

It demonstrates practical skills in **Excel data cleaning, reporting automation, dashboard design, and business analytics**.

## Key Features

### 📈 KPI Summary Cards
The dashboard tracks important business metrics such as:
 **Total Sales:** ₹75,01,77,492  
 **Total Boxes Sold:** 6,69,020  
 **Total Shipments:** 1,800  
 **Sales per Person:** ₹3,00,07,100  

### Sales by Geography
Interactive horizontal bar chart showing country-wise sales performance:
- India  
- Australia  
- USA  
- UK  
- Canada  
- New Zealand  

Used for regional comparison and market performance analysis.
### Sales by Category
Pie chart visualization representing contribution by product categories:
- Bars – 49%  
- Bites – 32%  
- Other – 19%  
Helps identify top revenue-generating categories.

###  Salesperson Performance Table
Detailed employee-wise report including:
- Salesperson Name  
- Sales Amount  
- Boxes Sold  
- Shipments Completed  
- Conditional Formatting Bars  
- Performance Indicators  

Used for tracking individual productivity.
